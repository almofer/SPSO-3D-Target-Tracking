# Speciated Particle Swarm Optimization for 3D Target Tracking

This repository contains a MATLAB implementation of Particle Swarm Optimization (PSO) for locating a moving target in a three-dimensional search space.

## Features

- Standard PSO framework
- Modular MATLAB implementation
- 3D visualization of particle movement
- Easily extensible for dynamic targets and multi-swarm variants

## Requirements

- MATLAB R2018a or newer

## Run

```matlab
main
```

## Repository Structure

- main.m
- initializeSwarm.m
- updateParticle.m
- costFunction.m
- createTarget.m
- plotSwarm.m
